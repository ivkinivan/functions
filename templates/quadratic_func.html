Квадратичная функция

Определение

Функция, заданная формулой вида y = ax² + bx + c (a ≠ 0), называется квадратичной.

a (старший коэффициент) — отвечает

за направление ветвей:

-если а > 0, то ветви параболы направлены вверх,

-если a < 0, то ветви параболы направлены вниз.

за ширину параболы:

-чем больше значение |а|, тем меньше ширина параболы.

b (второй коэффициент) — отвечает

за смещение параболы от начала координат.

c (свободный член) — соответствует

ординате точки пересечения параболы с осью Oy.


Исследование квадратичной функции

При a > 0

1. D(y)=R.

2. E(y)=R.

3. Функция не является ни четной, ни нечетной.

4. Нули функции x1 и x2 - корни уравнения ax² + bx + c = 0.

5. y > 0 при x∈(-∞;x1)U(x2; +∞).

ㅤy < 0 при x∈(x1;x2)

6. Возрастает на промежутке (хв; +∞) и убывает на промежутке (-∞; хв) .

(xв - абсцисса вершины параболы).

7. x min = хв. y min = y(xв).

8. Наименьшее значение функции y(хв), наибольшего значения функция не имеет.

При a < 0:

1. D(y)=R.

2. E(y)=R.

3. Функция не является ни четной, ни нечетной.

4. Нули функции x1 и x2 - корни уравнения ax² + bx + c = 0.

5. y > 0 при x∈(x1;x2).

ㅤy < 0 при x∈(-∞;x1)U(x2; +∞)

6. Возрастает на промежутке (-∞;хв) и убывает на (хв;+∞).

7. x max = хв; y max = y(x max).

8. Наибольшее значение функции y(хв), наименьшего значения функция не имеет.

Парабола в архитектуре.

Параболическая антенна - это тип антенны, который широко используется для направленных радиосвязей и радиолокации. Ее основным элементом является параллелепипедальная параболическая отражающая поверхность, которая фокусирует электромагнитные волны в пучок. Этот эффект позволяет увеличить дальность и точность связи.

Оптическое свойство параболы: все лучи, исходящие из источника света, находящегося в фокусе параболы, после отражения оказываются направленными параллельно ее оси. Это свойство используется при изготовлении прожекторов, автомобильных фар, карманных фонариков.

Интересный факт

В известном научно-фантастическом фильме СССР "Гиперболоид инженера Гарина" возникает сомнение в наличии самой конструкции этого фантастического оружия гиперболоида как поверхности вращения, а в фильме, показана конструкция, использующая эллиптический параболоид, так что с точки зрения математики фильм назван неверно.
