Квадратичная функция

Определение

Функция, задаваемая формулой y = ax² + bx + c (a ≠ 0), называется квадратичной. a — старший коэффициент. Отвечает за ширину параболы. Чем больше значение |а|, тем меньше ширина параболы. b — второй коэффициент. Отвечает за смещение параболы от начала координат. c — свободный член, соответствует ординате точки пересечения параболы с осью oy.

исследование функции

При a>0

1. D(y)=R.

2. E(y)=R.

3. Четность, нечетность. Функция не является ни четной, ни нечетной

4. Нули функции: нули функции - корни уравнения ax² + bx + c = 0

5. Промежутки знакопостоянства. Функция будет положительной на промежутках, где (a > 0) и отрицательной, где (a < 0).

y > 0 при x∈(-∞;x1)U(x2; +∞). y < 0 при x∈(x1;x2)

5. Промежутки монотонности. Если a>0, то функция убывает на промежутке (-∞;хв) и возрастает на (хв;+∞). (xв - х вершины параболы)

6.Точки экстемума. x min = 0; y min = y(x min) = 0

7. Наибольшие и наименьшие значения min y(x) = 0

Применение в жизни

Примеры применения квадратичной функции в жизни:

Зависимость перемещения тела от времени при равноускоренном движении прямо пропорционально квадрату времени движения.

Параболическая антенна - это тип антенны, который широко используется для направленных радиосвязей и радиолокации. Ее основным элементом является параллелепипедальная параболическая отражающая поверхность, которая фокусирует электромагнитные волны в пучок. Этот эффект позволяет увеличить дальность и точность связи.

Оптическое свойство параболы: все лучи, исходящие из источника света, находящегося в фокусе параболы, после отражения оказываются направленными параллельно ее оси. Это свойство используется при изготовлении прожекторов, автомобильных фар, карманных фонариков.
