Тригонометрическая функция

Основная информация

Функция заданная формулой вида y = sin(x) / y=cos(x) / y=tg(x) / y=ctg(x)

исследование функции

y = sin(x)

1. Область определения – x ∈(− ∞, + ∞).

2. Множество значений: E(y)=[-1;1].

3. Если sin(-x) = -sin(x) и D(y) симметрична относительно 0, то функция нечетная

4. sin(2π+x) = sin(x) - функция переодическая с периодом 2π

5. Нули функции: πn; n∈Z; sin(x)=0; x=πn n∈Z

6. Промежутки знакопостоянства y>0 при x∈(0+2πn; 2π+2πn) n∈Z

7. Промежутки монотонности

возрастает на [π/2+2πn;-π/2+2πn] n∈Z

убывает на [π/2+2πn;3π/2+2πn] n∈Z

8. Точки экстремумы и экстремумы функции x макс=π/2+2πn; n∈Z

y макс = 1 sin(x)=1 x=π/2+2πn; n∈Z

x мин=-π/2+2πn; n∈Z

y = cos(x)

1. D(y)=R.

2. E(y)=[-1;1].

3. Функция четная.

4. cos(2π+x) = cos(x) - функция переодическая с периодом 2π

5. Функция принимает значение, равное 0 при x=π/2+πn; n∈Z равное 1 при x=2πn; n∈Z

6. Промежутки знакопостоянства y>0 при x∈(-π/2+2πn; π/2+2πn) n∈Z

7. Промежутки монотонности.

возрастает на [π+2πn; 2π+2πn]; n∈Z

убывает на [0+2πn; 2π+2πn]; n∈Z

8. Точки экстремумы и экстремумы функции x макс=2π+2πn; n∈Z

y макс = 1

y мин = -1

sin(x)=1

x=π/2+2πn; n∈Z

x мин=π+2πn; n∈Z

y = tg(x)

1. D(y)=R. x≠π/2+πn; n∈Z

2. E(y)=R.

3. Функция нечетная.

4. tg(π+x) = tg(x) - функция переодическая с периодом π.

5. нули функции. y = 0 при x=πn; n∈Z

6. Промежутки знакопостоянства

y > 0 при x∈(-π/2+πn; π/2+πn) n∈Z

y < 0 при x∈(-π/2+πn; π/2+πn) n∈Z

7. Промежутки монотонности

возрастает на [-π/2+πn; π/2+πn] n∈Z на всей D(y) => функция возрастающая

8. Точек экстремум и экстремумов функции не имеет, наибольших и наименьших значений не имеет

y = сtg(x)

Применение в жизни

В физике — для описания колебательных процессов различной физической природы.

Тригонометрические функции используются для анализа звуковых волн, создания музыкальных композиций, проектирования инструментов, а также в обработке аудио и создании спецэффектов в кино и музыкальных произведениях.
