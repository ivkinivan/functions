Тригонометрическая функция

Определение

Функция заданная формулой вида y = sin(x) / y=cos(x) / y=tg(x) / y=ctg(x)

Исследование тригонометрических функций

y = sin x

1. D(y)=R.

2. E(y)=[-1;1].

3. Функция нечетная

4. sin(2π+x) = sin x - функция периодическая с основным периодом 2π

5. Нули функции: πn; n∈Z

6. Промежутки знакопостоянства

ㅤy>0 при x∈(0+2πn; π+2πn) n∈Z

ㅤy<0 при x∈(π+2πn; 2π+πn) n∈Z

7. Промежутки монотонности

ㅤвозрастает на [-π/2+2πn;π/2+2πn] n∈Z

ㅤубывает на [π/2+2πn;3π/2+2πn] n∈Z

8. Точки экстремумы и экстремумы функции

x max=π/2+2πn, y max = 1; n∈Z

x min=-π/2+2πn, y min = -1 ; n∈Z

y = cos x

1. D(y)=R.

2. E(y)=[-1;1].

3. Функция четная.

4. cos(2π+x) = cos x - функция периодическая с основным периодом 2π

5. Нули функции: π/2+πn; n∈Z.

6. Промежутки знакопостоянства:

ㅤy>0 при x∈(-π/2+2πn; π/2+2πn) n∈Z

ㅤy<0 при x∈(π/2+2πn; 3π/2+2πn) n∈Z

7. Промежутки монотонности:

ㅤубывает на [0+2πn; 2π+2πn]; n∈Z

ㅤвозрастает на [π+2πn; 2π+2πn]; n∈Z

8. Точки экстремумы и экстремумы функции:

ㅤx max=2πn; n∈Z, y max = 1

ㅤx min=π+2πn; n∈Z, y min = -1

y = tg x

1. D(y)=R. x≠π/2+πn; n∈Z

2. E(y)=R.

3. Функция нечетная.

4. tg(π+x) = tg x - функция периодическая с основным периодом π.

5. Нули функции: πn; n∈Z

6. Промежутки знакопостоянства:

ㅤy > 0 при x∈(0+πn; π/2+πn) n∈Z.

ㅤy < 0 при x∈(π/2+πn; π+πn) n∈Z.

7. Промежутки монотонности:

возрастает на [-π/2+πn; π/2+πn] n∈Z => функция возрастающая.

8. Точек экстремума и экстремумов функции не имеет.

9. Наибольшего и наименьшего значения функция не имеет.

y = сtg x

1. D(y)=R; x≠πn; n∈Z

2. E(y)=R.

3. Функция нечетная.

4. ctg(π+x) = tg x - функция периодическая с основным периодом π.

5. Нули функции. π/2+πn; n∈Z

6. Промежутки знакопостоянства:

ㅤy > 0 при x∈(0+πn; π/2+πn) n∈Z;

ㅤy < 0 при x∈(π/2+πn; π+πn) n∈Z.

7. Промежутки монотонности:

возрастает на [π/2+πn; π+πn] n∈Z.

убывает на [0+πn; π+πn] n∈Z.

8. Точек экстремума и экстремумов функции не имеет.

9. Наибольшего и наименьшего значения функция не имеет.

Применение в жизни

В физике — для описания колебательных процессов различной физической природы, напирмер, колебания маятника.

Тригонометрические функции используются для анализа звуковых волн, создания музыкальных композиций, проектирования инструментов, а также в обработке аудио и создании спецэффектов в кино и музыкальных произведениях.
